# bdavinfo
Vesti La Giubba氏によるbadvinfoのミラー。

badvinfoはBDAVディスクの情報をCSV出力するツールです。
ブルーレイレコーダーで記録したブルーレイディスクの内容を管理できます。

公開ページが既に閉鎖済みなので公開しました。
Windows版です。macOS版は含みません。
